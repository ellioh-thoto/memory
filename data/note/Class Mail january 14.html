Here is a summary of the USPS APIs update that will be effective on January 26 and that may affect v1.1

It will be available on January 13 on the test environment (and we should make a sanity test to ensure that we are safe).

 

Based on official release note published here: https://www.usps.com/business/web-tools-apis/2014-january-webtools-release-notes.rtf

 

eVS API

·         New Mail Classes

o   PARCEL SELECT NPS

o   MEDIA

o   LIBRARY

o   BPM

·         New Extra Services

o   Return Receipt

o   Return Receipt Electronic

o   Restricted Delivery

o   Adult Signature

o   Adult Signature Required Delivery

o   10:30 AM Delivery Option (for Express only)

o   Support of Hazardous Materials, Cremated Remains, and Live Animals (extra fields)

·         Support of COMMERCIAL PLUS pricing

·          (labels updated to take in account all mentioned new features)

 

Rate API

·         New Mail Class

o   First-Class Metered Mail

·         Service Information : text returned regarding additional guarantee information (should be displayed in step 3 ?)

·         As expected, No Commercial Plus pricing support

·         No BPM pricing support

 

Tracking API
